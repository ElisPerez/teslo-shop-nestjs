# Teslo API with NestJS

# Pasos:

1. Installation

```bash
$ yarn install
```

2. Configurar variables de entorno.
   - Clonar el archivo `.env.template` y renombrarlo a `.env`

3. Levantar DB PostgreSQL con docker-compose

```
docker-compose up -d
```

4. Running the app

```bash
# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
